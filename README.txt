# Combat Bets - version complète (simplifiée front)
Site de pari pour combats Minecraft (backend Node + sqlite3). Front livré en HTML/CSS/JS (Tailwind CDN).

## Installation
1. Dézippez
2. `npm install`
3. `npm start`
4. Ouvrez http://localhost:3000

## Notes
- Utilise sqlite3 (pas besoin de compiler native si binaire précompilé pour votre Node).
- Le front est simple mais soigné (Tailwind CDN).
- Les mots de passe sont stockés en clair (version demo). Pour production, hashez-les!
